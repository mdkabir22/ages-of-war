# Bug Triage Template

Use this template during Phase C validation runs.

## Severity Rules

- **P0 (Blocker)**: Game cannot continue, crash, data corruption, major progression break.
- **P1 (High)**: Core gameplay loop broken, mission logic wrong, severe balance exploit.
- **P2 (Medium)**: Noticeable issue with workaround; does not block full session.
- **P3 (Low)**: Cosmetic, text mismatch, minor UI friction.
